# Frontend Playground

A small sandbox for front-end experiments and notes. Most files are drafts
from coursework and self-study, kept here for personal reference.

## Structure

- `src/` — small JS snippets and toy components
  - `utils.js` — random collection of helper functions (WIP)
  - `dom.js` — DOM manipulation experiments
- `notes/` — markdown study notes
- `index.html` — quick scratch page

## Note

This is a personal learning repo. Nothing here is production-ready.
Feel free to ignore.
